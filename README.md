# 🌤️ WeatherWise: Real-Time Weather Dashboard

A sleek, responsive weather application built with vanilla web technologies. Get instant weather updates for any city and stay prepared with a comprehensive 5-day forecast.



## 🚀 Features

* **Real-Time Data:** Accurate current temperature, humidity, and wind speed.
* **5-Day Forecast:** Daily weather outlook including high/low temperatures and weather conditions.
* **Smart Search:** Find weather data for any city worldwide.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
* **Dynamic Backgrounds:** (Optional: mention if your UI changes based on weather conditions like Rain/Sun).
  
## 🛠️ Built With

* **HTML5:** Semantic structure for the dashboard.
* **CSS3:** Custom styling with Flexbox/Grid and Glassmorphism effects.
* **JavaScript (ES6+):** Fetch API for real-time data handling and DOM manipulation.
* **OpenWeatherMap API:** The engine providing the global weather data.

## 📦 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
You will need an API Key from [OpenWeatherMap](https://openweathermap.org/api).
1. Sign up for a free account.
2. Navigate to the **API Keys** tab and copy your unique key.

### Installation
1. Clone the repo:
   ```bash
   git clone [https://github.com/your-username/weather-app.git](https://github.com/your-username/weather-app.git)
2. Navigate to the project directory:
    ```bash
    cd weather-app
    ```
3. Open script.js and replace 'YOUR_API_KEY' with your actual key:
    ```bash
    const apiKey = 'YOUR_API_KEY_HERE';
    ```
4. Open index.html in your browser.
